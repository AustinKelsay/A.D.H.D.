# ADHD MVP Phase (Phase 3)

## Objective
Ship the first complete user loop: dictate -> conductor delegates -> worker executes -> summarized result.

## In Scope
- End-to-end run path for desktop and phone
- Live job state stream
- Approval and interrupt controls
- Final summary and artifact links

## Out of Scope
- Advanced analytics and enterprise controls

## Work Items
1. Unified intake UX
- Voice and text submission on desktop/phone.

2. Live execution view
- Show conductor + worker progress with clear state badges.

3. Human-in-the-loop controls
- Approval/rejection, interrupt, retry.

4. Result packaging
- Final summary with key outcomes and links to logs/diffs.

## Exit Criteria
- A real coding task can complete end-to-end from both desktop and phone.
- Approval and interrupt work reliably during active runs.
- Completed jobs are queryable with summary + artifacts.
